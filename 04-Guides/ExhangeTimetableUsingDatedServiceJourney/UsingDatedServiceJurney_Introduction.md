# Introduction
This guide describe the content of the shared file within zip file with timetable data which contain one shared file and one per line.

In this guide, the exhange of data is in one zip file with one shared file with common information, and one file per line:
- One shared file named _<CODESPACE> eg. "_shared_data.xml "
- One per line:  <COESPACE>_<LINE NAME AND DESCRIPTION>, eg. "RUT_RUT-Line-4_4_Vestli---Bergkrystallen.xml"
